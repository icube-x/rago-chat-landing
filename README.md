# KL-Store Landing Page

KL-Store 서비스의 랜딩 페이지입니다.

## 기술 스택

- React + TypeScript
- Vite
- Tailwind CSS v4
- Lucide Icons

## 설치 및 실행

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev

# 빌드
pnpm build
```

## 특징

- 반응형 디자인
- 5가지 요금제 (Trial, Starter, Pro, Business, Enterprise)
- "준비중입니다" 모달 기능
- 깔끔하고 현대적인 UI

## 배포

Vercel, Netlify 등의 플랫폼에 배포 가능합니다.
